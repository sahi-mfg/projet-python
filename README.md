# Projet Python

# Utilisation

- Cloner ce repo: `git clone https://github.com/sahi-mfg/projet-python`
- se placer dans le dossier: `cd projet-python`
- installer les packages (requests, sqlalchemy, python-dotenv et flask) (faire uv sync si vous utilisez uv)
- créer un fichier .env pour y mettre le token (ACCESS_TOKEN = ...)
- lancer le fichier main.py (`uv run main.py` si utilise uv ou `python main.py`)
- lancer l'API (`uv run api.py` ou `python api.py`)
